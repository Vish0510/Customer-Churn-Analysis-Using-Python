# 📊 Customer Churn Analysis (EDA) Using Python

This project performs **Exploratory Data Analysis (EDA)** on customer data to understand the key factors influencing customer churn.

---

## 🎯 Objective

- Analyze customer behavior  
- Identify patterns leading to churn  
- Help businesses reduce customer loss  

---

## 🛠 Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 📂 Dataset

The dataset includes:
- Customer demographics  
- Account information  
- Service usage  
- Churn status (Yes/No)  

---

## 📊 Analysis Performed

- Data Cleaning & Preprocessing  
- Handling missing values  
- Exploratory Data Analysis (EDA)  
- Data Visualization  

---

## 📈 Key Insights

- 📉 Customers with month-to-month contracts churn more  
- 💰 Higher monthly charges increase churn rate  
- ⏳ Long-term customers are less likely to churn  
- 📞 Customers without tech support churn more
